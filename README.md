# mobile-core-api

This package contains the mobile-core interfaces for mobile-modules

## Installation

```sh
yarn mobile-core-api
```

## Usage

```js
import { getStorageItem } from "mobile-core-api";

// ...

const result = await getStorageItem('ItemKeySample');
```
