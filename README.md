# mobile-core-api

This package contains the mobile-core interfaces for mobile-modules

## Installation

```sh
npm install mobile-core-api
```

## Usage

```js
import MobileCoreApi from "mobile-core-api";

// ...

const result = await MobileCoreApi.multiply(3, 7);
```
